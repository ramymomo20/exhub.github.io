# exhub.github-io

Initial research notes for the IOSCA Hub build are in [docs/research/iosca-hub-research.md](docs/research/iosca-hub-research.md).

Frontend app source is in [frontend](./frontend). Run it with `npm install` then `npm run dev` from that folder.

GitHub Pages deployment is handled by [deploy-pages.yml](./.github/workflows/deploy-pages.yml). In the repository settings, Pages should be set to `GitHub Actions` as the source.
